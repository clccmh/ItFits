It Fits

By: Carter Hay


Type of Game and Tools

  • It Fits is a puzzle game written with the tool Torque2D

Setup:

  • In order to play It Fits, navigate to the 'src' folder, and open Torque2D.exe.

Playing:

  • The object of It Fits is to place all of the shapes from the pile on the right, onto the board on the left.

  • At the top right corner, there are three lifelines, that can only be used once in the game. These are:

    • Expand the board
    • Delete one random block
    • Skip the level

  • To move the shapes, simply drag your mouse from the pile over to the board. It Fits will give you a shape from the pile, and the shapes will snap to the grid on the board as you drag them.

  • In order to win levels in It Fits, you should try to minimize the spaces on the board that cannot be reached. This is similar to the mechanic in the game Tetris.

  • In It Fits, there are 5 levels. While every level can be completed without lifelines, it is easiest to complete the game if you use one lifeline on level 3, and another lifeline on level 5.

  • If you cannot complete the game, or get stuck on a level, plese reference this video. Please note, that the color of peices, and order of peices is randomized.
